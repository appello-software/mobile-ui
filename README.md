# @appello/mobile-ui

Appello Mobile UIKit.

# Install

With `npm`:

```
npm install @appello/mobile-ui react-hook-form react-native-svg

npx expo install expo-linear-gradient
```

With `yarn`:

```
yarn add @appello/mobile-ui react-hook-form react-native-svg

npx expo install expo-linear-gradient
```
